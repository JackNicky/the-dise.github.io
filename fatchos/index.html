    How to install resources?

    • Download and install Notify & Fitness (8.13.1);

    • Download the resources of the desired version. We look at the correspondence of the firmware version and resources according to the table;

    • Launch Mi Fit, synchronize and minimize the application;

    • Open Notify & Fitness. Go to "Settings" - "Basic" - "Mi Band firmware version" - "Update" - select the file - select the previously downloaded resources and click "Update"

      • IMPORTANT!

      When the program writes 100%, this does not mean that the resources are installed. Wait for the bracelet to restart.

    How to determine the version of the bracelet?

    The visible difference is that in the version with NFC, the arc bent upwards is shown at the bottom of the screen, (U) if there is a circle there without NFC.(O)
